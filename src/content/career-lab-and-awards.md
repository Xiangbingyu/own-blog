# 校园实验室与竞赛征途：技术成长的加速器

作为杭州电子科技大学的一名计算机专业学生，我始终坚信“以赛促学”是技术成长的最佳路径。在校期间，我深度参与了多个高水平竞赛与实验室项目，这些经历不仅打磨了我的技术栈，更培养了解决复杂工程问题的能力。

## 核心竞赛经历

### 阿里云 Data+AI 工程师全球大奖赛
- **项目**：DeepResearch Agent 框架
- **角色**：核心开发者
- **成果**：搭建了一个具备自主规划能力的 DeepResearch Agent。该 Agent 能够拆解复杂问题，调用 MCP 搜索引擎获取背景信息，通过多轮循环搜索与验证，最终生成高质量的研究报告。这一过程加深了我对 **Agent 编排**与 **Function Call** 的理解。

### Soul App Agent 创新赛 & 易班创新技术大赛
- **项目**：幻境协创 (Fantasy Co-creation Platform)
- **角色**：全栈开发 & AI 架构设计
- **成果**：入围百强 Agent 并获得易班创新奖。
  - **痛点解决**：针对用户构建世界观难的问题，开发了基于 Function Call 的引导工具，一键生成结构化设定。
  - **技术亮点**：设计了上下文管理机制，确保 AI 角色扮演的沉浸感；利用 **CrewAI** 搭建多智能体工作流（分析-风格-写作），实现了从大纲到章节的自动化生成。

### Global Game Jam (全球游戏制作节)
- **体验**：在 48 小时极限开发中，与团队协作完成游戏原型。这段经历极大地锻炼了我的 **Unity** 开发能力与团队沟通效率。
