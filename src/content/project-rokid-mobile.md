# Rokid Mobile System (慧视校园)

专为 Rokid AR 眼镜设计的手机端配套应用，通过手机与眼镜的联动，赋能智慧校园场景。

## 核心场景

- **AR 导览**：校园地标识别与增强信息展示。
- **辅助教学**：虚拟教具展示与互动。
- **即时通讯**：基于 AR 视角的远程协作与指导。

## 技术实现

- **多屏互动**：手机作为交互手柄与计算单元，眼镜作为显示终端。
- **计算机视觉**：集成物体识别与空间定位算法。
- **Android 原生开发**：深度调用系统能力，优化性能与功耗。
