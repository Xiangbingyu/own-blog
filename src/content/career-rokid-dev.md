# Rokid AR 眼镜开发实录：Android 原生与多模态交互

在 AI 思政实验室期间，我主导了 Rokid AR 眼镜相关应用的开发。这是一个将 Android 原生技术与 AI 能力结合的典型软硬件协同项目。

## 核心挑战与突破

### 1. 双端通信与协同
项目要求手机端与眼镜端实时同步信息。
- **解决方案**：基于 Android 原生 Socket 通信与 Rokid SDK 的互联协议，实现了低延迟的数据传输。例如，眼镜端拍摄的照片实时传输至后端服务器进行人脸识别，识别结果毫秒级回传至手机与眼镜屏幕。

### 2. “随心一听”智能音乐助手
这是一个独立开发的创意功能，旨在通过视觉感知用户情绪。
- **技术链条**：
    1.  **端侧采集**：调用 Rokid Camera SDK 拍摄第一视角照片。
    2.  **多模态分析**：上传照片至后端，结合用户填写的 MBTI 与心情标签，利用多模态大模型分析当前场景氛围。
    3.  **推荐算法**：后端返回适配的音乐列表。
    4.  **原生播放**：使用 Kotlin 开发了完整的音乐播放器 UI 与后台服务。

### 3. 技术栈拓展
为了完成该项目，我从零开始系统学习了 **Kotlin** 语言与 **Android Studio** 开发环境，并深入研读了 Rokid 官方文档，掌握了 AR 设备特有的交互逻辑开发。
