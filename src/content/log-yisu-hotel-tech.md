# 易宿酒店开发实录：JWT、高德地图与阿里云短信

在开发 `Yisu-Hotel` 酒店预订平台的过程中，为了提升系统的安全性和用户体验，集成了多项第三方服务与认证机制。

## 关键技术点

### JWT (JSON Web Token) 身份认证
放弃了传统的 Session 认证，采用 JWT 实现无状态的身份验证。这使得后端服务更易于水平扩展，同时也方便了跨域请求的处理。

### 高德地图 API 接入
为了让用户能直观地查看酒店位置，接入了高德地图 SDK。
- **定位服务**：自动获取用户当前城市。
- **周边检索**：展示酒店附近的交通与设施。

### 阿里云短信服务
集成阿里云 SMS 服务用于用户注册与登录的验证码发送。
- 封装了统一的短信发送工具类。
- 实现了验证码的缓存与过期机制（Redis）。
