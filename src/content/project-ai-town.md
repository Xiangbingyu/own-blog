# AI Town

一个由 AI 驱动的虚拟小镇，基于 Unity 和 Python 构建，旨在探索多智能体在虚拟环境中的自主交互与社会涌现。

## 核心特性

- **智能决策系统**：NPC 基于当前状态、记忆和环境感知生成行动计划。
- **自然语言交互**：角色间基于性格与背景进行有意义的对话。
- **记忆与反思**：具备长期记忆，能够从过往经历中学习并影响未来决策。
- **环境感知**：实时感知周围环境与其他角色的动态。

## 技术架构

- **前端 (Unity/C#)**：负责场景渲染、角色动画与寻路系统。
- **后端 (Python)**：集成大语言模型 (LLM)，处理决策逻辑与对话生成。
- **通信机制**：通过文件监控与 HTTP 请求实现前后端数据同步。
